== JavaCC/JJTree Skeleton Project ==

JavaCC/JJTree project with grammar for skeleton DSL called MD

To run: 

  mvn exec:java

See ./src/main/java/com/zippyrate/dsls/md/parser/package-info.java
for description of handling of manually-modified AST JJTree node classes.

Also see MDDumpVisitor.java in that directory for a very basic visitor class.
